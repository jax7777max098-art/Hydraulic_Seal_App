   flutter config --android-sdk "C:\Android\Sdk"

2. قم بالبناء:

   flutter pub get
   flutter build apk --release

   build/app/outputs/flutter-apk/app-release.apk


للكمبيوتر (Windows EXE):
───────────────────────────────────────────────────────────────
1. تأكد من تثبيت Visual Studio Community مع C++

2. قم بالبناء:
   cd C:\Users\Abo-Osama\Desktop\Hydraulic_Seal_Logger
   flutter pub get
   flutter build windows --release

3. الملف النهائي:
   build/windows/runner/Release/hydraulic_seal_logger.exe


للويب (Web Version):
───────────────────────────────────────────────────────────────
1. أضف دعم الويب (إذا لم يكن موجوداً):
   flutter create . --platforms web

   flutter build web --release

📊 خصائص البيانات المخزنة
════════════════════════════════════════════════════════════════

كل سجل يحتوي على:
  • ID: معرف فريد
  • machineName: اسم الآلية (Volvo, CAT, إلخ)
  • machineType: نوع الطقم (Bobcat Kit, Boom Kit, إلخ)
  • dateInput: تاريخ الإدخال
  • mainLeakSeal: جلدة تهريب رئيسية
  • leakBackUpSeal: سنادة تهريب
  • hbySeal: جلدة HBY
  • dustArea: مساحة غبرة
  • neckSeal1: سيلة رقبة 1
  • neckSeal2: سيلة رقبة 2
  • neckSealBackup: سنادة سيلة رقبة
  • releaseSeal: جلدة تريح
  • graphiteSeal: سكمان غرافيت
  • fiberCenter: سنطرة فيبر
  • innerPistonSeal: سيلة بسطون داخلية
  • pistonSealBackup: سنادة سيلة بسطون
