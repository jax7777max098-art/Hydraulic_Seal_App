# Hydraulic Seal Logger - تسجيل قياسات الأختام الهيدروليكية

تطبيق Flutter كامل لتسجيل وإدارة قياسات الأختام الهيدروليكية، يعمل على Windows و Android و iOS.

## الميزات الرئيسية

### 📱 واجهة مستخدم
- **Dark Mode** احترافي مع ألوان زرقاء
- **خطوط كبيرة** وواضحة لسهولة القراءة
- **تمييز بصري** للسيلات (أزرق) والسنادات (أحمر)
- **تصميم متجاوب** يعمل على جميع الشاشات

### 🗄️ إدارة البيانات
- **قاعدة بيانات محلية** (SQFlite)
- **إضافة وتعديل وحذف** الأطقم
- **عرض الأطقم** في كروت منظمة
- **بحث وفلترة** الأطقم

### 📊 المقارنة الذكية
- **مقارنة متعددة** بين الأطقم
- **تلوين ذكي** للقيم المختلفة
- **عرض الفرق** بوضوح
- **تصدير نتائج المقارنة**

### 💾 التصدير والاستيراد
- **تصدير إلى Excel** (.xlsx)
- **استيراد من Excel**
- **نسخ احتياطي** للبيانات
- **مشاركة البيانات** بين الأجهزة

### 🔧 التقنيات المستخدمة
- **Flutter** - إطار العمل الرئيسي
- **Dart** - لغة البرمجة
- **SQFlite** - قاعدة البيانات
- **Excel** - معالجة ملفات Excel
- **File Picker** - اختيار الملفات
- **Shared Preferences** - التخزين المحلي

## البنية التقنية

```
lib/
├── main.dart                 # الملف الرئيسي والصفحات
├── db/
│   └── database_helper.dart  # إدارة قاعدة البيانات
└── models/

```

## التشغيل والتطوير

### متطلبات النظام
- Flutter SDK 3.0+
- Dart SDK
- Android Studio أو VS Code

### خطوات التشغيل
```bash
# تحميل المكتبات
flutter pub get

# تشغيل على المحاكي
flutter run

# البناء للمنصات المختلفة
flutter build windows  # للكمبيوتر
flutter build apk      # للأندرويد
flutter build ios      # للـ iOS
```

## الاستخدام

### إضافة طقم جديد
1. اضغط على "إضافة طقم جديد"
2. أدخل اسم الطقم والقياسات
3. احفظ الطقم

### عرض الأطقم
- عرض جميع الأطقم في الصفحة الرئيسية
- البحث بالاسم
- عرض التفاصيل بالضغط على الطقم

### المقارنة
1. اختر "مقارنة الأطقم"
2. حدد الأطقم المراد مقارنتها
3. عرض النتائج مع التلوين

### التصدير
1. اذهب إلى صفحة التصدير
2. اختر الأطقم المراد تصديرها
3. حدد مسار حفظ الملف
